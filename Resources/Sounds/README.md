# Sounds

Drop the AAC sound assets here as `.m4a` files. They are bundled with
each app target via Xcode's Resources phase.

## Required for v1.0

End sounds (ship with free tier):
- `end-chime.m4a` — soft mallet on metal, 1.2 s decay
- `end-bell.m4a` — small ceremonial bell, 1.0 s decay

White noise pack (Pro tier, downloaded on demand):
- `wn-lightRain.m4a`
- `wn-heavyRain.m4a`
- `wn-wind.m4a`
- `wn-ocean.m4a`
- `wn-forestMorning.m4a`
- `wn-coffeeShop.m4a`
- `wn-libraryHum.m4a`
- `wn-trainCabin.m4a`
- `wn-nightCrickets.m4a`
- `wn-brownNoise.m4a`
- `wn-brownLowRumble.m4a`
- `wn-pinkNoise.m4a`
- `wn-tapeHissVinyl.m4a`
- `wn-singingBowl.m4a`
- `wn-distantChimes.m4a`

See `docs/05-sound.md` for sourcing strategy and the 1.2-second rule.
